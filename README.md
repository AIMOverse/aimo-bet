# AiMo Chat

## Description

An open-sourced AI Chat UI powering aimo.chat - uncensored, private AI, developed by AiMo Network

## Design Philosophy

- Bring your own key (BYOK)
- Clean Dependencies based on most developer-friendly sdk (vercel-based as priority)
  - nextjs
  - tailwind css
  - shardcn
  - ai-elements
  - ai-sdk
  - use workflow
  - x402-enabled
- Any models, tools, or agents
- Clean, modular UI
- Multi-platform (pwa, mobile, desktop) via tauri (planned)
